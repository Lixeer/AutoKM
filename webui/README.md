# AutoKM WebUI
作者：[WindAFeng](https://github.com/WindAFeng)

该模板可帮助你快速使用 Vue 3 和 Vite 开始开发。

本项目编译配置中已包含对于单文件输出插件的配置

## 推荐的 IDE 设置

[VS Code](https://code.visualstudio.com/) + [Vue 官方插件](https://marketplace.visualstudio.com/items?itemName=Vue.volar)（请禁用 Vetur）。

## 项目配置

详见 [Vite 配置参考](https://vite.dev/config/)。


### 开发模式（带热重载）

```sh
npm run dev
```

### 生产构建（含类型检查、编译和压缩）

```sh
npm run build
```
